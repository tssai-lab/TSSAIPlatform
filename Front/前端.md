### 技术选型

*   **框架:** React 18+
*   **脚手手架:** Vite (比Create React App更快更现代)
*   **UI 库:** Ant Design 5.x (专业、强大的企业级UI解决方案)
*   **路由:** React Router DOM 6.x (实现页面跳转)
*   **图标库:** @ant-design/icons (配合Ant Design使用)

### 第一步：创建项目并安装依赖

首先，打开您的终端，执行以下命令来创建一个新的React项目。

```bash
# 1. 使用 Vite 创建项目 (my-ai-platform 是你的项目名)
npm create vite@latest my-ai-platform -- --template react

# 2. 进入项目目录
cd my-ai-platform

# 3. 安装所需的核心依赖
npm install antd react-router-dom @ant-design/icons
```

---

### 第二步：规划项目文件结构

一个清晰的文件结构是良好规范的开始。在 `src` 目录下，我们创建如下结构：

```
my-ai-platform/
└── src/
    ├── assets/               # 存放静态资源, 如 logo.svg
    ├── components/           # 可复用的公共组件
    │   └── AppHeader/        # 统一的顶部栏组件
    │       ├── index.jsx
    │       └── style.css
    ├── pages/                # 页面级组件
    │   ├── Login/            # 登录页
    │   │   ├── index.jsx
    │   │   └── style.css
    │   └── Main/             # 主界面
    │       ├── index.jsx
    │       └── style.css
    ├── routes/               # 路由配置
    │   └── index.jsx
    ├── App.jsx               # 应用根组件，用于配置路由
    ├── main.jsx              # 应用入口文件
    └── index.css             # 全局样式
```

#### 2. 配置应用路由 (`src/routes/index.jsx` 和 `src/App.jsx`)

**`src/routes/index.jsx` (新建文件):**
```jsx
import { useRoutes, Navigate } from 'react-router-dom';
import LoginPage from '../pages/Login';
import MainPage from '../pages/Main';

export default function AppRouter() {
  const routes = useRoutes([
    {
      path: '/login',
      element: <LoginPage />,
    },
    {
      path: '/main/*', // 使用通配符*，允许MainPage内部有自己的嵌套路由
      element: <MainPage />,
    },
    {
      path: '/',
      element: <Navigate to="/login" replace />, // 默认重定向到登录页
    },
  ]);
  return routes;
}
```

**`src/App.jsx` (修改文件):**
```jsx
import { ConfigProvider } from 'antd';
```

#### 4. 实现登录页 (`src/pages/Login/`)

**`src/pages/Login/index.jsx` (新建文件):**
```jsx
import React, { useState } from 'react';
import { Layout, Card, Form, Input, Button, Tabs, message } from 'antd';
import { UserOutlined, LockOutlined } from '@ant-design/icons';
import { useNavigate } from 'react-router-dom';
import AppHeader from '../../components/AppHeader';
import './style.css';

const { Content } = Layout;

const LoginPage = () => {
  const navigate = useNavigate();
  const [activeTab, setActiveTab] = useState('login');

  const onFinish = (values) => {
    console.log('Received values of form: ', values);
    // 模拟登录/注册成功
    message.success(`${activeTab === 'login' ? '登录' : '注册'}成功！`);
    // 跳转到主界面
    navigate('/main');
  };

  const LoginForm = (
    <Form name="login" onFinish={onFinish}>
      <Form.Item name="username" rules={[{ required: true, message: '请输入用户名!' }]}>
        <Input prefix={<UserOutlined />} placeholder="用户名" />
      </Form.Item>
      <Form.Item name="password" rules={[{ required: true, message: '请输入密码!' }]}>
        <Input.Password prefix={<LockOutlined />} placeholder="密码" />
      </Form.Item>
      <Form.Item>
        <Button type="primary" htmlType="submit" block>
          登录
        </Button>
      </Form.Item>
    </Form>
  );

  const RegisterForm = (
    <Form name="register" onFinish={onFinish}>
      <Form.Item name="username" rules={[{ required: true, message: '请输入用户名!' }]}>
        <Input prefix={<UserOutlined />} placeholder="设置用户名" />
      </Form.Item>
      <Form.Item name="password" rules={[{ required: true, message: '请输入密码!' }]}>
        <Input.Password prefix={<LockOutlined />} placeholder="设置密码" />
      </Form.Item>
       <Form.Item name="confirm" dependencies={['password']} hasFeedback rules={[
          { required: true, message: '请确认您的密码!' },
          ({ getFieldValue }) => ({
            validator(_, value) {
              if (!value || getFieldValue('password') === value) {
                return Promise.resolve();
              }
              return Promise.reject(new Error('两次输入的密码不匹配!'));
            },
          }),
        ]}>
        <Input.Password prefix={<LockOutlined />} placeholder="确认密码" />
      </Form.Item>
      <Form.Item>
        <Button type="primary" htmlType="submit" block>
          注册
        </Button>
      </Form.Item>
    </Form>
  );

  const tabItems = [
    {
      key: 'login',
      label: `账号密码登录`,
      children: LoginForm,
    },
    {
      key: 'register',
      label: `注册账号`,
      children: RegisterForm,
    },
  ];

  return (
    <Layout className="login-layout">
      <AppHeader />
      <Content className="login-content">
        <Card className="login-card">
          <Tabs activeKey={activeTab} onChange={setActiveTab} items={tabItems} centered />
        </Card>
      </Content>
    </Layout>
  );
};

export default LoginPage;
```

#### 5. 实现主界面 (`src/pages/Main/`)

**`src/pages/Main/index.jsx` (新建文件):**
```jsx
import React, { useState } from 'react';
import { Layout, Menu } from 'antd';
import {
  HistoryOutlined,
  ApartmentOutlined,
  DatabaseOutlined,
} from '@ant-design/icons';
import AppHeader from '../../components/AppHeader';
import './style.css';

const { Sider, Content } = Layout;

// 这可以是从路由动态生成的，这里为简化直接定义
const menuItems = [
  { key: 'history', icon: <HistoryOutlined />, label: '训练记录' },
  { key: 'models', icon: <ApartmentOutlined />, label: '我的模型' },
  { key: 'data', icon: <DatabaseOutlined />, label: '我的数据' },
];

// 定义每个菜单项对应的右侧内容组件
const contentMap = {
  history: () => <div>这里是训练记录的内容区</div>,
  models: () => <div>这里是我的模型的内容区</div>,
  data: () => <div>这里是我的数据的内容区</div>,
};


const MainPage = () => {
  const [selectedKey, setSelectedKey] = useState('history'); // 默认选中项

  const renderContent = () => {
    const Component = contentMap[selectedKey];
    return Component ? <Component /> : <div>请选择一个菜单项</div>;
  }

  return (
    <Layout className="main-layout">
      <AppHeader />
      <Layout>
        <Sider width={200} className="main-sider">
          <Menu
            mode="inline"
            selectedKeys={[selectedKey]}
            onClick={({ key }) => setSelectedKey(key)}
            style={{ height: '100%', borderRight: 0 }}
            items={menuItems}
          />
        </Sider>
        <Layout className="main-content-layout">
          <Content className="main-content">
            {renderContent()}
          </Content>
        </Layout>
      </Layout>
    </Layout>
  );
};

export default MainPage;
```

### 第四步：启动项目

回到终端，确保您在项目根目录 (`my-ai-platform`)下，然后运行：

```bash
npm run dev
```

Vite会启动开发服务器，您可以在浏览器中打开 `http://localhost:5173` (或终端提示的地址) 查看效果。

### 最终效果

1.  **访问应用**：默认进入登录页，有登录和注册两个Tab。
2.  **登录**：输入任意内容，点击“登录”，会提示成功并自动跳转到主界面。
3.  **主界面**：
    *   顶部是统一的 `AppHeader`。
    *   左侧是带图标的Sider菜单。
    *   点击左侧不同的菜单项（训练记录、我的模型、我的数据），右侧内容区会相应地切换显示文本。

这个架构清晰、组件化，并且易于扩展。您可以轻松地在 `contentMap` 中替换为更复杂的组件，或者进一步使用嵌套路由来管理更复杂的页面状态。
